API

  • 方法介绍
  • tap手指点击触发
  • swipestart手指触摸的时候触发
  • swipeend手指松开的时候触发
  • swipeleft向左滑动，手指松开后触发,执行一次
  • swiperight向右滑动，手指松开后触发,执行一次
  • swipetop向上滑动，手指松开后触发,执行一次
  • swipedown向下滑动，手指松开后触发,执行一次
  • holdleft向左滑动，手指移动的时候触发,一直执行
  • holdright向右滑动，手指移动的时候触发,一直执行
  • holdtop向上滑动，手指移动的时候触发,一直执行
  • holddown向下滑动，手指移动的时候触发,一直执行
  • holdx横向移动，手指移动的时候触发,一直执行
  • holdy竖向移动，手指移动的时候触发,一直执行
  • hold手指移动的时候触发,一直执行

  var rtouch = new Rtouch(obj); 
  rtouch.tap = function(e){ ... };
  rtouch.swipeleft = function(e){ ... };   
  ....
  rtouch.hold = function(e){ ... };

配置

  • 配置介绍
  • SWIPE_TRESHOLD数值型 默认20 — 最少滑动距离
  • PREDEF数组型 默认[1,0]
    [X轴移动是否取消默认，Y轴移动是否取消默认](默认取消X轴的默认)
    — 阻止默认事件 一般用来阻止滚动条滚动 

  var rtouch = new Rtouch(obj,{
  	SWIPE_TRESHOLD : 50,    //最少滑动距离 改为50
  	PREDEF : [1,1]    //阻止X和Y轴的默认事件
  });

传递参数

  • 参数介绍
  • cX手指第一次触摸时的X轴的距离
  • cY手指第一次触摸时的Y轴的距离
  • x手指当前的X轴距离
  • y手指当前的Y轴距离
  • type当前的滑动状态 格式："swipeleft"
  • e默认的触摸对象
